# 健身记录导出应用

这是一个iOS应用，用于将Apple Health中的健身记录导出为CSV文件。

## 功能特点

- 📊 读取Apple Health中的健身数据
- 📝 导出为标准的CSV格式
- ⏰ 包含开始时间、结束时间、持续时间
- 🏃‍♂️ 识别不同的健身类型（跑步、步行、骑行等）
- 🔥 包含卡路里消耗和距离数据
- 📱 简洁的用户界面

## 使用方法

### 1. 打开项目

1. 确保您已安装Xcode（推荐版本14.0或更高）
2. 双击 `WorkoutExport.xcodeproj` 文件打开项目
3. 等待Xcode索引完成

### 2. 配置项目

1. 选择您的开发团队：
   - 在Xcode中，选择 `WorkoutExport` 项目
   - 在 `Signing & Capabilities` 标签页
   - 选择您的Apple开发者账号
   - 如果没有开发者账号，可以选择个人团队（需要登录Apple ID）

2. 配置Bundle Identifier（可选）：
   - 默认Bundle Identifier为 `com.example.WorkoutExporter`
   - 您可以修改为唯一的标识符

### 3. 运行应用

1. 选择目标设备：
   - 连接您的iPhone到电脑
   - 或者在模拟器中运行

2. 点击运行按钮（▶️）或按 `Cmd + R`

3. 首次运行时需要授权HealthKit权限：
   - 应用会请求访问健身数据的权限
   - 请允许所有请求的权限

### 4. 导出数据

1. 打开应用后，点击"导出CSV"按钮
2. 等待数据加载和导出完成
3. 导出完成后，可以选择分享CSV文件：
   - 通过AirDrop发送到其他设备
   - 保存到文件应用
   - 通过邮件发送

## 导出的CSV格式

CSV文件包含以下列：
- 开始时间：健身活动的开始时间
- 结束时间：健身活动的结束时间  
- 持续时间(分钟)：活动的总时长
- 健身类型：活动类型（跑步、步行、骑行等）
- 卡路里(千卡)：消耗的卡路里
- 距离(米)：运动距离
- 平均心率：平均心率（需要额外权限）

## 注意事项

- 需要iOS 16.0或更高版本
- 需要iPhone设备（不支持iPad）
- 需要Apple Health中有健身记录数据
- 首次使用需要授予HealthKit数据访问权限

## 技术支持

如果遇到任何问题：
1. 确保已授予HealthKit权限
2. 检查iPhone是否已登录iCloud
3. 确认Apple Health中有健身记录数据

## 隐私说明

本应用仅读取健身数据用于导出，不会上传或分享您的任何健康数据。所有数据处理都在设备本地完成。
